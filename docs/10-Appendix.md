# 10 – Appendix

## Glossary

- **VLAN** — Virtual LAN for traffic isolation  
- **Honeypot** — decoy system to attract and study attackers  
- **Tar-pit** — slows attacker interactions intentionally  
- **IDS / IPS** — Intrusion Detection / Prevention System  
- **IaC** — Infrastructure as Code, declarative provisioning/management  
- **Snapshot** — saved state of VM/container for rollback
